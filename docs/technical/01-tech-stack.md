# 기술 스택

## Frontend (앱)

| 항목 | 선택 | 이유 |
| --- | --- | --- |
| Framework | Flutter 3.x | 크로스플랫폼, 학습곡선 완만 |
| State Management | Riverpod | 안정적, 테스트 용이 |
| Local Storage | Hive | 오프라인 캐싱 |

## Backend

| 항목 | 선택 | 이유 |
| --- | --- | --- |
| BaaS | Firebase | 서버리스, 빠른 구축 |
| Auth | Firebase Auth | 소셜 로그인 (Google, Apple) |
| Database | Firestore | 실시간 동기화 |
| Push | Cloud Messaging | FCM |
| Storage | Cloud Storage | 몬스터 이미지 저장 |
| Functions | Cloud Functions | 주간 리포트 생성 |

## 기타

| 항목 | 선택 |
| --- | --- |
| Analytics | Firebase Analytics |
| Crash Report | Firebase Crashlytics |
| CI/CD | GitHub Actions + Fastlane |
| Design | Figma |

## 선택 이유

- 혼자 개발하기에 최적
- 인증, DB, 푸시, 스토리지 다 해결
- 무료 티어로 MVP 충분
- 취업용 포트폴리오로도 좋음 (Flutter 수요 많음)
