# 기록 시스템 설계

## 기록 방식

### 플로우

```
1. 카테고리 선택 (탭)
      ↓
2. 소분류 선택 (탭)
      ↓
3. 심각도 슬라이더 (1~5, 선택)
      ↓
4. 한 줄 메모 (선택)
      ↓
5. 완료 → 몬스터 획득 연출
```

### 심각도 레벨

| 레벨 | 설명 | 경험치 보너스 |
| --- | --- | --- |
| 1 | 살짝 | x1.0 |
| 2 | 약간 | x1.1 |
| 3 | 보통 | x1.2 |
| 4 | 많이 | x1.3 |
| 5 | 극심 | x1.5 |

## 일일 기록 제한

| 항목 | 설정 |
| --- | --- |
| 하루 최대 | 5회 |
| 카테고리 중복 | 허용 |
| 6회 이상 시도 | "오늘 너무 열심히 퇴화 중... 내일 또 봐요" |

---

## 검증 방식: 반신뢰 시스템

### 원칙

- 기록은 자유롭게 (신뢰 기반)
- 단, 솔직할수록 희귀 몬스터 획득
- "거짓말해도 본인 손해" 구조

### 장치들

| 장치 | 설명 |
| --- | --- |
| 상세 기록 보너스 | 메모 작성 시 경험치 1.5배 |
| 패턴 기반 희귀 | 3일 연속 같은 퇴보 → 진화형 해금 |
| 랜덤 양심 체크 | 가끔 "진짜야?" 팝업 (심리적 압박) |
| 자기 기록 회고 | 주간 리포트에서 본인 기록 다시 보여줌 |

## 카테고리 설계

### 대분류 → 소분류

| 대분류 | 소분류 | 관련 몬스터 |
| --- | --- | --- |
| 🍖 식습관 | 야식, 폭식, 배달, 과식 | 야식귀신, 배달의기사 |
| 😴 수면 | 늦잠, 밤샘, 낮잠과다 | 이불요정, 올빼미 |
| 🏃 운동/건강 | 운동 스킵, 계단 안 감 | 소파감자, 엘베요정 |
| 💸 돈 | 충동구매, 과소비, 현질 | 텅장유령, 카드귀신 |
| 📱 생산성 | 미루기, SNS 과몰입, 유튜브 | 내일요정, 스크롤좀비 |
| 👥 관계 | 연락 씹음, 약속 펑크 | 읽씹유령, 잠수도깨비 |
| 🍺 습관 | 술, 담배, 카페인 | 주정령, 니코틴령 |
| ❓ 기타 | 자유 입력 | 미분류 몬스터 |

### MVP 카테고리 (6개)

1. 🍖 식습관
2. 😴 수면
3. 🏃 운동/건강
4. 💸 돈
5. 📱 생산성
6. ❓ 기타
