# 화면 플로우

## 탭 구조

```
┌─────────────────────────────────────┐
│                                     │
│            [콘텐츠 영역]              │
│                                     │
├─────────────────────────────────────┤
│  🏠 홈  │  ✏️ 기록  │  📖 도감  │  👤 마이  │
└─────────────────────────────────────┘
```

---

## 전체 플로우

```
[스플래시]
     ↓
[온보딩] ──→ (최초 1회만)
     ↓
[로그인]
     ↓
┌─────────────────────────────────────────┐
│              메인 (4탭)                  │
├─────────┬─────────┬─────────┬───────────┤
│   홈    │   기록   │   도감   │    마이    │
└────┬────┴────┬────┴────┬────┴─────┬─────┘
     │         │         │          │
     ↓         ↓         ↓          ↓
  [홈 화면]  [기록플로우]  [도감]    [마이]
     │                    │          │
     │                    ↓          ├→ [주간 리포트]
     │              [몬스터 상세]     ├→ [업적]
     │                              └→ [설정]
     │
     ├→ [주간 리포트 배너] (월요일)
     └→ [FAB] → 기록 플로우
```

---

## 화면별 상세

### 1. 스플래시

- 앱 로고 + 로딩
- 2초 내외

### 2. 온보딩 (최초 1회)

| 화면 | 내용 |
| --- | --- |
| 온보딩 1 | "오늘 뭐 망했어?" 컨셉 소개 |
| 온보딩 2 | 기록하면 몬스터 생김 설명 |
| 온보딩 3 | 도감 수집 설명 |

### 3. 로그인

- Google 로그인 (MVP)
- Apple 로그인 (iOS)

## 홈 탭

### 레이아웃

```
┌─────────────────────────┐
│  🔥 7일 연속 퇴화 중!    │  ← 스트릭
├─────────────────────────┤
│  📊 1월 1주차 리포트!    │  ← 월요일만 노출
│     MVP: 야식귀신 🏆      │
├─────────────────────────┤
│                         │
│      [대표 몬스터]       │  ← 탭하면 도감으로
│       야식귀신           │
│      "오늘도 치킨?"      │  ← 랜덤 한마디
│                         │
├─────────────────────────┤
│  오늘 퇴보 (2/5)        │
│  🍖 야식  😴 늦잠        │
├─────────────────────────┤
│       [+ 기록하기]       │  ← FAB 또는 버튼
└─────────────────────────┘
```

### 구성 요소

| 요소 | 설명 | 탭 액션 |
| --- | --- | --- |
| 스트릭 | 연속 기록 일수 | - |
| 리포트 배너 | 월요일만 노출, 확인하면 사라짐 | 리포트 화면으로 |
| 대표 몬스터 | 가장 많이 소환한 몬스터 | 도감 상세로 |
| 오늘 퇴보 | 오늘 기록 목록 (5개 제한) | - |
| FAB | 기록 바로가기 | 기록 플로우로 |
