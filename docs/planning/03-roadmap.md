## Phase 1: MVP (17일)

**목표**: 스토어 출시 가능한 최소 버전

### 구현 범위

- ✅ C1. 퇴보 기록
- ✅ C2. 몬스터 생성 (10종)
- ✅ C3. 도감 (기본)
- ✅ C4. 홈 대시보드
- ✅ R1. 스트릭
- ✅ T1. 로그인 (Google만)
- ✅ T2. 프로필 기본
- ❌ 푸시 알림 (로컬 알림만)
- ❌ 주간 리포트
- ❌ 소셜 기능 전체
- ❌ 수익화 전체

### 일정

| 일차 | 작업 | 산출물 |
| --- | --- | --- |
| D1 | 프로젝트 세팅, Firebase 연동 | 빈 앱 + 인증 동작 |
| D2 | Firestore 스키마, 기록 CRUD | 기록 저장/조회 |
| D3 | 기록 화면 UI | 카테고리 선택 + 심각도 + 메모 |
| D4 | 몬스터 로직 (해금 조건, 경험치) | 기록 → 몬스터 생성 |
| D5 | 도감 화면 | 목록 + 상세 |
| D6 | 홈 대시보드 | 오늘 기록, 스트릭, 최근 몬스터 |
| D7 | 마이페이지 + 설정 | 프로필, 알림 설정 |
| D8~9 | 전체 연결 + 플로우 점검 | 앱 동작 완성 |
| D10~11 | 몬스터 이미지 적용 + UI 다듬기 | 비주얼 완성 |
| D12~14 | **QA + 버그 수정** | 테스트 케이스 소화 |
| D15~17 | **스토어 준비 + 심사 제출** | 스크린샷, 설명, 심사 |
