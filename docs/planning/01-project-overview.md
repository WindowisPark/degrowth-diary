# 프로젝트 개요

## 한 줄 요약

나쁜 습관 기록하면 몬스터가 생기는 역설적 자기계발 앱

## 슬로건 후보

- "퇴화해서 진화한다"
- "오늘도 당당하게 망했다"
- "나의 흑역사 도감"

## 핵심 컨셉

> 모든 인간에겐 '그림자 도감'이 있다.
>
> 우리가 미루고, 먹고, 눌러버린 모든 순간들...
>
> 그 틈새에서 몬스터가 태어난다.
>
> 그들은 나쁜 존재가 아니다.
>
> 그냥 우리의 솔직한 민낯일 뿐.
>
> 자, 오늘은 어떤 녀석이 깨어날까?

## 핵심 루프

```
퇴보 기록 → 포인트 적립 → 몬스터 해금/성장 → 도감 수집 → 또 기록하고 싶어짐
```

## 타겟 유저

### 1차 타겟

**20대 중후반 직장인/취준생**

- 자기관리 압박 있는데 잘 안 됨
- 자조적 유머 좋아함
- "난 원래 이래" 마인드

### 2차 타겟

**10대 후반 ~ 20대 초반**

- 수집 요소에 반응
- SNS 공유 가능성 높음

## 차별화 포인트

| 기존 습관앱 | 퇴화일기 |
| --- | --- |
| 잘하면 보상 | 못해도 보상 |
| 실패 = 죄책감 | 실패 = 수집 |
| 스트레스 | 재미 |

**핵심**: 죄책감 없이 기록 → 기록 습관 → 자기 패턴 인식 → 실제 개선
