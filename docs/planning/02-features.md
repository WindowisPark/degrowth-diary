## Phase 요약

### Phase 1 (MVP)
- C1~C4: 핵심 루프 (기록 → 몬스터 → 도감)
- R1: 스트릭
- T1~T2: 인증, 프로필

### Phase 2
- C5: 푸시 알림
- R2, R4, R5: 리텐션 강화
- S1: 공유
- T3~T4: 설정 확장

### Phase 3
- R3: 월간 리포트
- S2~S4: 소셜 기능
- M1~M3: 수익화
- T5: 다크모드
