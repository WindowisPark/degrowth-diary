# 몬스터 비주얼 컨셉 가이드

## 🎨 전체 디자인 방향

### 핵심 키워드
- **귀엽지만 조금 음침한 (Cute but Gloomy)**
- **친근하지만 괴생명체 (Friendly but Alien)**
- **2D 일러스트 스타일**
- **다크 배경에서 돋보이는 컬러**

### 참고 레퍼런스
- **Pokémon**: 진화 시스템, 속성별 디자인
- **Don't Starve**: 다크하지만 귀여운 아트 스타일
- **Cult of the Lamb**: 악마적이지만 사랑스러운 캐릭터
- **Hollow Knight**: 어둡지만 아름다운 생명체 디자인

## 📐 기본 디자인 원칙

### 1. 실루엣 우선
- **가로 60-80px, 세로 60-100px** 크기에서도 식별 가능해야 함
- 각 몬스터는 고유한 실루엣을 가져야 함
- 진화할수록 복잡해지는 실루엣

### 2. 표정 시스템
모든 몬스터는 3가지 표정을 가진다:
- 😊 **기본 (Idle)**: 평화로운 표정
- 😋 **기쁨 (Happy)**: 새로운 기록이 생겼을 때
- 😴 **졸림 (Sleepy)**: 오래 방치되었을 때

### 3. 애니메이션
최소한의 애니메이션:
- **Idle**: 제자리에서 살짝 흔들림 (2프레임 루프)
- **Spawn**: 처음 나타날 때 페이드인
- **Evolution**: 진화할 때 빛 이펙트

### 4. 컬러 팔레트

| 속성 | 기본 색상 | 보조 색상 | 배경 강조 색 |
|------|-----------|-----------|--------------|
| Food (식습관) | `#FF6B6B` 빨강 | `#FFA07A` 연어 | `#FF4757` |
| Sleep (수면) | `#A29BFE` 보라 | `#6C5CE7` 진보라 | `#5F27CD` |
| Exercise (운동) | `#00D2D3` 청록 | `#01A3A4` 진청록 | `#0ABDE3` |
| Money (소비) | `#FFD93D` 금색 | `#FFC312` 진금색 | `#F79F1F` |
| Productivity (생산성) | `#6BCB77` 초록 | `#4D96A9` 청록 | `#05C46B` |

## 🦖 카테고리별 디자인 컨셉

### 1. Food Type (식습관) - 빨강 계열

**컨셉**: 귀엽고 통통하지만 약간 게걸스러운

#### 야식이 (Stage 1) - Common
```
모양: 작은 구형 생명체
특징:
  - 작은 몸통에 큰 입
  - 짧은 팔다리 (4개)
  - 입에서 침 흘리는 모습
  - 머리에 치킨 뼈다귀 장식
크기: 60x60px
색상: 밝은 빨강 + 분홍 볼
표정: 항상 배고픈 표정 (>_<)
```

#### 야식귀신 (Stage 2) - Rare
```
모양: 커진 몸체 + 유령 같은 형태
특징:
  - 반투명한 몸체 (고스트 효과)
  - 손에 배달 앱 들고 있음
  - 몸 주변에 피자/치킨 조각 떠다님
  - 야행성 느낌의 짙은 눈 밑
크기: 70x80px
색상: 어두운 빨강 + 보라 그라데이션
표정: 새벽의 유혹 표정 (ㅇㅅㅇ)
```

#### 야식대마왕 (Stage 3) - Epic
```
모양: 거대한 몸집 + 왕관
특징:
  - 배가 냉장고처럼 생김
  - 여러 개의 팔 (음식을 동시에 먹음)
  - 머리에 음식 왕관
  - 몸에 야식 메뉴판 문신
크기: 80x100px
색상: 진한 와인 레드 + 금색 액센트
표정: 만족스러운 포식자 표정 (^ㅁ^)
```

### 2. Sleep Type (수면) - 보라 계열

**컨셉**: 몽환적이고 피곤해 보이는

#### 늦잠이 (Stage 1) - Common
```
모양: 구름 같은 둥근 형태
특징:
  - 항상 눈 감고 있음
  - 머리에 알람시계 장식
  - 몸 주변에 'Z' 표시
  - 아래쪽이 살짝 녹아내리는 형태
크기: 60x65px
색상: 연한 라벤더 + 흰색 하이라이트
표정: 졸린 표정 (=_=)
```

#### 불면령 (Stage 2) - Rare
```
모양: 수직으로 늘어진 유령형
특징:
  - 피곤한 눈 밑 다크서클
  - 손에 스마트폰 들고 있음
  - 몸에서 빛나는 화면 빛
  - 밤하늘 패턴 (별과 달)
크기: 65x85px
색상: 진보라 + 네온 블루 (스마트폰 빛)
표정: 또렷하지만 피곤한 표정 (o_o)
```

#### 수면파괴자 (Stage 3) - Epic
```
모양: 악마 같은 거대 생명체
특징:
  - 머리에 뿔 (알람시계 모양)
  - 날개 (하지만 너무 피곤해서 못 펼침)
  - 몸에 커피잔과 에너지드링크 문양
  - 주변에 카페인 오라
크기: 80x95px
색상: 다크 퍼플 + 핑크 에너지 이펙트
표정: 각성했지만 좀비 같은 표정 (O_O)
```

### 3. Exercise Type (운동) - 청록 계열

**컨셉**: 게으르고 축 늘어진

#### 나태충 (Stage 1) - Common
```
모양: 애벌레 같은 형태
특징:
  - 배를 땅에 붙이고 누워있음
  - 짧고 무기력한 팔다리
  - 머리에 소파 쿠션 모자
  - 몸에 '내일부터' 글자
크기: 70x50px (가로로 길쭉)
색상: 민트 그린 + 회색 음영
표정: 완전 무기력 (u_u)
```

#### 게으름보 (Stage 2) - Rare
```
모양: 나무늘보 같은 형태
특징:
  - 소파에 녹아붙어 있음
  - 한 손에 리모컨
  - 다른 손에 과자봉지
  - 몸 주변에 먼지 쌓임
크기: 75x70px
색상: 청록 + 베이지 (소파 색)
표정: "귀찮아..." 표정 (-_-)
```

#### 만년소파인 (Stage 3) - Epic
```
모양: 소파와 융합된 거대 생명체
특징:
  - 몸이 소파 자체가 됨
  - 여러 팔로 동시에 게임/먹방/넷플릭스
  - 헬스장 회원권이 먼지 쌓여서 화석화
  - 주변에 게으름 오라 (다른 몬스터도 졸게 만듦)
크기: 90x80px
색상: 다크 틸 + 갈색 (낡은 소파)
표정: 만족스러운 게으름 (^_^)zzz
```

### 4. Money Type (소비) - 금색 계열

**컨셉**: 반짝이고 유혹적인

#### 충동이 (Stage 1) - Common
```
모양: 동전 모양의 둥근 생명체
특징:
  - 큰 눈 (반짝반짝)
  - 작은 날개 (천사 같지만 악마)
  - 손에 카드 들고 있음
  - "이건 사야 해!" 말풍선
크기: 60x60px
색상: 밝은 금색 + 흰색 반짝임
표정: 순진한 척하는 표정 (◕‿◕)
```

#### 과소비령 (Stage 2) - Rare
```
모양: 쇼핑백을 뒤집어쓴 유령
특징:
  - 몸 주변에 쇼핑백과 택배 상자
  - 손에서 카드 광선
  - 머리에 "Sale" 표시
  - 주변에 영수증 떠다님
크기: 70x80px
색상: 금색 + 오렌지 그라데이션
표정: "어머 이건 사야 해" 표정 (owo)
```

#### 통장귀신 (Stage 3) - Epic
```
모양: 거대한 악마 형태
특징:
  - 몸이 텅 빈 지갑 모양
  - 여러 손으로 동시에 쇼핑
  - 머리에 카드/현금/계좌이체 왕관
  - 주변에 "-1000원", "-50000원" 이펙트
크기: 85x95px
색상: 다크 골드 + 빨강 (마이너스)
표정: 사악한 미소 (▽)
```

## ✨ 희귀도별 이펙트

### Common (일반) - 희귀도 없음
```
- 기본 컬러만 사용
- 단순한 Idle 애니메이션
- 이펙트 없음
```

### Rare (희귀) - 파란 빛
```
- 몸 테두리에 살짝 빛남 (파란색)
- Idle 시 반짝이는 별 파티클 (3초마다)
- 획득 시 파란 빛 원형 이펙트
```

### Epic (영웅) - 보라 빛
```
- 몸 전체에 보라색 오라
- Idle 시 빛나는 파티클 (1초마다)
- 획득 시 보라 빛 폭발 이펙트
- 배경에 미세한 빛 굴절
```

### Legendary (전설) - 금색 빛
```
- 몸 전체에 금색 후광
- Idle 시 지속적인 반짝임
- 획득 시 화면 전체 금색 플래시
- 배경이 일시적으로 밝아짐
- 특수 등장 애니메이션 (하늘에서 내려옴)
```

```

  - 천장에서 떨어지는 습관 에너지 (입자)

색상:
  - 배경: #2C2C3A (다크 그레이)
  - 바닥: #1E1E2E (더 어두운 그레이)
  - 액센트: #4A4A5A (약간 밝은 그레이)
```

- 몬스터들이 기지개

```

#### 저녁 (18:00-23:00)
```

## 🎬 애니메이션 가이드

### 기본 애니메이션

#### Idle (대기)

## 📦 에셋 제작 가이드

### 파일 포맷
- **원본**: SVG (벡터) 또는 PSD (레이어 분리)
- **게임용**: PNG (투명 배경)
- **해상도**: 3배수 (@1x, @2x, @3x)

### 레이어 구조 (PSD)
```
몬스터명 (예: 야식이)
├── Shadow (그림자)
├── Body (몸통)
│   ├── Main Color
│   └── Shading
├── Face (표정)
│   ├── Eyes
│   └── Mouth
├── Accessories (장식)
└── Effects (이펙트)
```

### 사이즈 가이드
- **Common**: 60x60px ~ 70x70px
- **Rare**: 65x75px ~ 75x80px
- **Epic**: 80x90px ~ 90x100px
- **Legendary**: 90x100px ~ 100x110px

### 색상 프로필
- **Color Space**: sRGB
- **Bit Depth**: 32-bit (RGBA)
